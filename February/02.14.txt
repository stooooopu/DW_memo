3가지
1. position
	1-1. static : 디폴트값 안움직임
	1-2. relative : 본인 자리 기준으로 움직임
	1-3. absolute : 부모태그(relative라면) 기준으로 움직임
	1-4. fixed : 스크롤따라 내려옴
2. padding margin box-size
	2-1. padding : 부모와 자식태그 간격 조절
	2-2. margin : 부모태그끼리 간격 조절
	2-3. box-size : border-box
		선 밖으로 튀어나가지 않게

3. flex (inline-block)
	default = inline-block
	container와 item으로 구성 (배열)
	container의 길이와 item 간격을 유연하게 조절
	☆★ex) display : flex
